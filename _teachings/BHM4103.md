**Programme:** Bachelor of Mechatronic Engineering Technology / Bachelor of Mechatronic Engineering

## Course Description

This course covers the analysis and design of feedback control systems for engineering applications. Students gain both theoretical understanding and practical skills in designing stable and responsive controllers.

## Topics Covered

- System modelling and transfer functions
- Time and frequency domain analysis
- Stability analysis (Routh-Hurwitz, Bode, Nyquist)
- PID controller design and tuning
- Root locus techniques
- State-space representation

## Learning Outcomes

By the end of this course, students will be able to:
1. Model dynamic systems using transfer functions and state-space representations
2. Analyse system stability using time and frequency domain methods
3. Design PID controllers to meet specified performance requirements
4. Apply root locus and frequency response methods for controller design
5. Implement and test control systems using simulation tools
