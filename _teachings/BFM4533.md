**Course Code:** BFM4533

**Programme:** Bachelor of Mechatronic Engineering Technology

**Institution:** Universiti Malaysia Pahang Al-Sultan Abdullah (UMPSA)

## Course Description

This course equips students with the skills to design, fabricate, and test functional robotic prototypes. Emphasis is placed on the integration of mechanical, electronic, and software subsystems into a complete working robot.

## Topics Covered

- Robot mechanical design and CAD modelling
- Actuator and sensor selection and integration
- Robot control system implementation
- Prototype fabrication techniques
- System testing, evaluation, and documentation

## Learning Outcomes

By the end of this course, students will be able to:
1. Design a complete robotic system from concept to prototype
2. Select and integrate appropriate actuators and sensors for a given application
3. Implement a control system for a robotic prototype
4. Fabricate, test, and evaluate a functional robotic prototype
5. Document and present engineering design work professionally
