**Programme:** Bachelor of Mechatronic Engineering Technology (Robotics)

**Institution:** Universiti Malaysia Pahang Al-Sultan Abdullah (UMPSA)

## Course Description

This course introduces students to the mathematical and computational modelling of robotic systems. Students develop skills in kinematic and dynamic analysis, enabling them to model, simulate, and evaluate robot behaviour.

## Topics Covered

- Spatial descriptions and transformations
- Forward and inverse kinematics
- Denavit-Hartenberg (DH) parameter convention
- Robot dynamics and equation of motion
- Trajectory planning and motion simulation
- Introduction to robot programming and simulation tools (MATLAB Robotics Toolbox, ROS)

## Learning Outcomes

By the end of this course, students will be able to:
1. Apply spatial transformation mathematics to describe robot configurations
2. Derive and solve forward and inverse kinematics for serial manipulators
3. Use the DH convention to model robot arm geometry
4. Simulate robot motion using computational tools
5. Plan and evaluate robot trajectories for given tasks
